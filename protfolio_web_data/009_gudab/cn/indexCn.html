gudab :

企业级MongoDB监控管理平台

MongoDB是企业最普遍使用的NoSQL数据库，但若缺乏适当的监控管理工具将会使企业在系统营运过程无法实时掌控与解决问题，进而隐藏管理风险。gudab提供了企业所需的管理功能，包含效能指针、监控、报警以及备份恢复等，让企业在使用MongoDB时能满足全方位的管理要求。

功能特色

集群架构自动侦测

Auto-discovery自动侦测并描绘系统拓朴，将所有集群内MongoDB服务器自动列出，并显示主机型态与列表以便设定进行监控。

指标监控

可针对CPU、RAM、Connection、Disk、Index、Throughput等超过30种以上指标进行监控，透过实时图表，显示系统状况。

系统报警

透过服务器清单以及监控指针，管理者可个别针对服务器设定所有监控指标之报警触发动作，支持E-mail与SNMP等报警通知方式。

MongoDB系统活动日志

记录受监控之MongoDB服务器的活动历程，以利稽核工作

个性化仪表板

提供管理者针对监控指针自行设定仪表板组合，以利日常监控管理工作。也可透过我的最爱方式将监控指针加入清单。

备份与恢复

可进行差异备份以及汇出与汇入恢复，并可进阶设定监控目标。

产品优势

针对企业管理需求

完整企业管理所需功能，包含系统仪表板、监控与报警、备份恢复、权限管理等，并提供系统规划建置等专业服务，让您使用MongoDB获得最完整的支持。

无需安装Agent程式

采用 "访问式" 取得各项指针和数据，无须在被监控端MongoDB另外安装探针，大幅减少系统运维上的困扰。

浏览器操作

透过浏览器即可操作，能让使用者于远程监控 MongoDB Server。

完整监控指标

提供超过30种指标， 提供全方位各式指针数据以供管理人员了解MongoDB运行状况。

最快时间更新

Stay up-to-date with MongoDB 。gudab支持最新MongoDB版本，从 version 3.4 开始以最快速度跟随MongoDB新版本的发行。

完整 MongoDB 专业服务

MongoDB 全方位服务，从数据库架构规划，实施，到异地备援，全备援，系统报警整合等专业顾问与技术支持。
