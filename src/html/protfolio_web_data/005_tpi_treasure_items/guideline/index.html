Overview

  • 本 guideline 為惜物網開發使用，以方便管理設計與 HTML/CSS 的整合和未來的延伸。
  • 在文件中會附上呈現畫面與原始碼，以便複製與比對。
  • 若有新增共用元件，也請協作者一併更新，讓未來參與的人員做為參考。

目錄

01. Colours 前往

網站所使用的顏色，與對應的樣式名稱

02. Typography 前往

與文字相關的設定放置在此處。如 h1 ~ h6、a、p...等文字標籤

03. Grid

本網站所使用的網格系統，以 Bootstrap v4.6 為基底做調整

04. Elements 前往

沒有和其他原件搭配的單一元件設定，如 button、input。

05. Components 前往

不同元件所構成的組件，使用時需注意層級與樣式，如 table。

使用方式

在 head 套用以下 CSS 檔

<link rel="stylesheet" href="../css/bootstrap.css">
<link rel="stylesheet" href="../css/sprite.css">
<link rel="stylesheet" href="../css/style.css">

在 body 最底部套用以下 JS 檔

<script src="../js/jquery-3.3.1.min.js"></script>
<script src="../js/popper.min.js"></script>
<script src="../js/bootstrap.min.js"></script>
<script src="../js/main.js"></script>
